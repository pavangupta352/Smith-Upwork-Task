Welcome to Extract!

Automatically extract data from your pdf files in bulk!
  1. Select the folders where the files to be extracted are saved (click on Add Folders above)
  2. Select the folder where you want the results to be saved
  3. Navigate to your files under Folders (above), and click on any of your files (it does not matter which file you start with) and mark it to start the process (see instructions below)
  4. Once a file is marked, the program will identify all the files that follow the same pattern as the file you marked and attempt to extract the data
  5. Successfully extracted files will disapear from the list (you can always get them back on the list if you wish, see View above)
  6. The extracted data will be saved in consolidated Excel files in the Results folder you selected

Mark your files

Extract uses AI and advanced algorithms to find the data in your files with your help.

Marking consists of defining landmarks in the text that the program uses to find the data.
For example, if your pdf file shows Account Number: 123456, and you want to capture the account number Click on "Acount Number" to create a marker that acts as a landmark and save it, then click on 123456 to create a marker that captures data specified by the landmark.
